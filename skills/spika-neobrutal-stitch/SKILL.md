# Spika Neo-Brutal Stitch Skill

Generate a `DESIGN.md` that another agent or Google Stitch can follow to create Spika-style Neo-brutalist interfaces.

## Output contract

Always output a complete `DESIGN.md` with these sections:

1. Visual theme and atmosphere.
2. Color palette and roles.
3. Typography rules.
4. Component styling.
5. Layout principles.
6. Motion and interaction.
7. Responsive rules.
8. Accessibility rules.
9. Anti-patterns.

## Style content to encode

- Paper background `#edeff1`.
- Ink text, border, and shadow `#1c1c1e`.
- Primary purple `#6200ea`.
- Accent surfaces `#ffe566`, `#7ee8fa`, `#ff8fab`, `#b8f397`.
- Space Mono UI labels.
- Inter and Pretendard Variable for body copy.
- 2px ink borders.
- 3px, 5px, and 6px hard offset shadows.
- Grid backgrounds and section separators.
- Tactile hover and active states.
- Visible focus rings.

## Anti-patterns to encode

- Glass panels.
- Soft shadows.
- Gradient mesh heroes.
- Rounded pill-only components.
- Thin gray structural borders.
- Random accent colors.
- Generic dashboard filler.
- Hidden focus states.
